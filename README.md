# МАТЕМАТИКА 2.0 — Сайт математического центра

Современный, адаптивный сайт для образовательного центра «МАТЕМАТИКА 2.0».

## Структура проекта

```
Лера сайт/
├── index.html              # Главная страница
├── css/
│   ├── style.css           # Основные стили
│   └── document.css        # Стили для страниц документов
├── js/
│   └── main.js             # JavaScript функциональность
├── pages/
│   ├── offer.html          # Публичная оферта
│   ├── privacy.html        # Политика конфиденциальности
│   └── refund.html         # Политика возврата
└── assets/
    └── images/             # Изображения (добавить)
```

## Функциональность

- Адаптивный дизайн (мобильные, планшеты, десктоп)
- Анимации при прокрутке (AOS)
- Слайдер отзывов
- Анимированные счётчики
- Плавающие элементы в hero-секции
- Фиксированная навигация
- Кнопка "наверх"
- Готовность к Яндекс.Картам

## Что нужно настроить перед публикацией

### 1. Заполнить данные ИП
В файлах `index.html`, `pages/offer.html`, `pages/privacy.html`, `pages/refund.html` найти и заменить:
- `[ФИО]` — фамилия, имя, отчество ИП
- `[номер]` — ИНН, ОГРНИП, расчётный счёт, БИК
- `[название банка]` — название банка

### 2. Добавить Яндекс.Карты
В `index.html` найти секцию `#contacts` и раскомментировать код карты, заменив `YOUR_API_KEY` на реальный ключ API:
```html
<div id="yandex-map" style="width: 100%; height: 400px;"></div>
<script src="https://api-maps.yandex.ru/2.1/?apikey=YOUR_API_KEY&lang=ru_RU"></script>
```

### 3. Добавить изображения (опционально)
Можно добавить реальные фотографии преподавателя и учеников в соответствующие секции.

### 4. Изменить контент
- Обновить цены в секции "Стоимость"
- Добавить реальные отзывы
- Обновить статистику (количество учеников, лет опыта)

## Инструкция по размещению на хостинге

### Шаг 1: Регистрация и покупка
1. Зарегистрируйтесь на выбранном хостинге
2. Купите домен (или подключите существующий)
3. Приобретите тарифный план хостинга

### Шаг 2: Загрузка файлов
1. Войдите в панель управления хостингом
2. Откройте файловый менеджер
3. Перейдите в папку `public_html` или `www`
4. Загрузите все файлы проекта (сохраняя структуру папок)

### Шаг 3: Настройка SSL
1. В панели найдите раздел "SSL-сертификаты"
2. Выберите бесплатный Let's Encrypt
3. Активируйте для вашего домена
4. Включите принудительный редирект на HTTPS

### Шаг 4: Проверка
1. Откройте сайт по вашему домену
2. Проверьте работу на разных устройствах
3. Убедитесь, что HTTPS работает (замок в адресной строке)

## Локальный запуск для тестирования

### Вариант 1: Python (если установлен)
```bash
cd "Лера сайт"
python -m http.server 8000
```
Откройте: http://localhost:8000

### Вариант 2: Node.js (если установлен)
```bash
npx serve
```

### Вариант 3: VS Code Live Server
1. Установите расширение "Live Server"
2. Правый клик на `index.html` → "Open with Live Server"

### Вариант 4: Просто откройте файл
Дважды кликните на `index.html` — откроется в браузере
